# SentinelOps Security Dashboard

A professional, modern DevSecOps security dashboard built with React and Flask. This dashboard provides comprehensive visualization and analysis of security scan results from Bandit (Python static analysis) and Trivy (container vulnerability scanning).

### Dashboard Overview
- **Security Score** - Overall security health grade (A+ to F)

- Python code security analysis results
- Filter by severity and confidence

### Trivy Container Scan
- Container vulnerability scanning results
- Package-level vulnerability breakdown
- CVE details with CVSS scores
- Fix version recommendations

### Charts & Visualizations
- **Pie Charts** - Severity and confidence distribution
- **Bar Charts** - Vulnerabilities by type/package
- **Area Charts** - Trend over time
- **Radar Charts** - Security posture overview

## ğŸ› ï¸ Installation

### Prerequisites
- Node.js 18+ and npm
- Python 3.10+
- Security scan reports in `reports/` directory

### Backend Setup

```bash
# Navigate to dashboard directory
cd dashboard

# Install Python dependencies
pip install -r requirements.txt

# Start the Flask backend
python dashboard-backend.py
```

The backend will start on `http://localhost:5000`

### Frontend Setup

```bash
# Navigate to frontend directory
cd dashboard/frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

The React app will start on `http://localhost:3000`

### Production Build

```bash
# Build for production
cd dashboard/frontend
npm run build

# The build output will be in 'dist' folder
npm run preview  # Preview production build
```

## ğŸ”§ Configuration

### Vite Proxy (Development)

The Vite dev server proxies API requests to Flask:

```javascript
// vite.config.js
server: {
  port: 3000,
  proxy: {
    '/api': {
      target: 'http://localhost:5000',
      changeOrigin: true,
    },
  },
}
```
